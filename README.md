# ⚡️ ⚡️ ⚡️ Introduction
Server side processes requests and real-time services (users send message, users are typing, ...)

[👉🏻 Client side](https://github.com/GiaKhangLuu/android-appchat)

## ⚙️ Technologies used ##

**DBMS:** MongoDB

**Framework:** ExpressJS

**Libraries:** SocketIO (for real-time service), ...
